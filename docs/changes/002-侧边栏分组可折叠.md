# 002 侧边栏分组可折叠

- 状态：已完成
- 日期：2026-06-01
- 关联提交：f0dfb09

## 需求 / 改动

左侧导航栏的"图纸核价 / 机加工挑图 / 系统"三组，改前组标题是静态灰字、组内项永远展开。希望点击组标题就能展开/收起该组内容，让侧边栏更干净、按需展开。

## 方案

三组统一可折叠（避免"有的能点有的不能点"的割裂感）。关键决策：

- **各自独立**开关，互不影响（不是手风琴）——流程间严格隔离，操作员会来回切，强制只开一组反而碍事。
- **默认只展开当前选中项所在那组**（即"图纸核价"，含默认选中的"待处理"），另两组收起，初始最干净。
- 组标题右侧加状态箭头：展开朝下 `ChevronDown`、收起朝右 `ChevronRight`。
- 收起的组里项被隐藏、点不到，所以选中逻辑无需"选中自动展开"。

## 实现

- `src/Draftline.App/ViewModels/ShellViewModel.cs`：`NavGroup` 改为 `ObservableObject`，加 `IsExpanded`（默认 true）；新增 `ToggleGroupCommand(NavGroup)`；构建后把第一组外的组 `IsExpanded=false`。
- `src/Draftline.App/Views/ShellWindow.xaml`：组标题由静态 `TextBlock` 换成 `GroupHeader` 样式按钮（标题 + 箭头，命令绑 `ToggleGroupCommand`）；组内项 `ItemsControl` 的 `Visibility` 绑 `IsExpanded`（`BoolToVis`）。
- `src/Draftline.App/Converters/Converters.cs` + `App.xaml`：新增 `BoolToChevronConverter`（注册为 `BoolToChevron`）。
- `src/Draftline.App/Styles/FluentTheme.xaml`：新增 `GroupHeader` 样式（无边框透明、左对齐、悬停淡底圆角）。

导航/选中/内容区/权限均未改动。

## 验证

`dotnet build src/Draftline.App -p:EnableWindowsTargeting=true` 0 错 0 警。WPF 真机折叠点击效果需在 Windows 上确认。
