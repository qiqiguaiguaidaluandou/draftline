# 001 批次列表"本地文件夹"名做成"打开表单"入口

- 状态：已完成
- 日期：2026-06-01
- 关联提交：f0dfb09

## 需求 / 改动

待处理 / 已处理批次列表里有一列"本地文件夹"，改前只是静态文本。希望点击文件夹名就能进入表单作业界面，效果与右侧"打开表单"按钮一致。

## 方案

把"本地文件夹"列从纯文本 `DataGridTextColumn` 改成可点链接，点击触发与"打开表单"按钮相同的 `OpenBatchCommand`（→ `_nav.ToBatchWork(...)`）。
- 文件夹名做成链接样式（强调蓝、悬停下划线、手型光标），并加 `ToolTip="打开表单作业"` 提示——因为"点文件夹=进表单"与直觉里"点文件夹=开目录"有张力。
- 右侧原有的文件夹图标按钮（在资源管理器打开目录）保持不动，三个入口各管各的：文件夹名 / 打开表单 = 进作业界面；文件夹图标 = 开本地目录。

## 实现

- `src/TZHJ.App/Views/BatchListView.xaml`：`本地文件夹` 列改为 `DataGridTemplateColumn`，单元格放一个 `Button`（`Style=BtnLink`、`Content={Binding FolderName}`、`Command=OpenBatchCommand`、`CommandParameter={Binding}`）。
- `src/TZHJ.App/Styles/FluentTheme.xaml`：新增 `BtnLink` 链接样式（无边框透明、强调色、悬停下划线）。

待处理 / 已处理共用同一个 `BatchListView`，两边同时生效。

## 验证

`dotnet build src/TZHJ.App -p:EnableWindowsTargeting=true` 0 错 0 警。WPF 真机点击效果需在 Windows 上 `dotnet run --project src/TZHJ.App` 确认。
