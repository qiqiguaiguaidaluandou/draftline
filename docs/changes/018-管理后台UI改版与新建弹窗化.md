# 018 管理后台 UI 改版：视觉美化 + 新建/编辑弹窗化

- 状态：已完成
- 日期：2026-06-30
- 关联提交：edbe7a3

## 需求 / 改动

Blazor 管理后台（`TZHJ.Gateway`，[012](012-Blazor管理后台.md)）此前样式偏朴素、平面，希望更**美观、丝滑**；另外"新建用户""新建角色"原本是**页面上常驻的表单卡片**，希望改成**点「新建」按钮后弹窗填写**。

两点合并为一次纯前端（CSS + Razor 组件）改版，不动任何服务端逻辑、权限模型与数据。

## 方案

- **视觉系统**：在 `App.razor` 内联样式表里改为 `:root` 设计令牌（配色/圆角/阴影/动效统一），整体仍**零外部依赖、纯内联 CSS**。所有类名保持不变，故各页面标记基本不动。
- **新建/编辑弹窗化**：新增**通用弹窗外壳** `Modal`（标题 + 内容 + 底部操作 + 关闭，`@bind-Visible` 双向绑定，复用模态样式），把两页的创建表单搬入弹窗；角色页"编辑"复用同一弹窗（一套表单兼顾新建/编辑）。

## 实现

1. `src/TZHJ.Gateway/Components/App.razor`（重写样式表）：
   - 设计令牌：靛蓝主色 `#4f46e5→#6366f1` 渐变、冷调中性灰底 `#f5f7fb`、状态色（绿/红/琥珀/靛）成体系、分层柔和阴影、圆角变量。
   - 侧栏深色竖向渐变 + 品牌色 logo 方块，选中项渐变胶囊带柔光、悬停轻微右移；卡片描边 + 柔阴影；表格行悬停高亮；按钮渐变+悬停上浮/按下回弹/焦点光环；输入聚焦主色描边+光环；徽标圆角胶囊化。
   - 动效（"丝滑"）：内容/卡片入场上浮淡入、提示条下滑、模态背景模糊+缩放淡入；美化滚动条；`prefers-reduced-motion` 下全部关闭。
   - 新增 `.page-head`（页头：标题 + 右侧主操作）、`.modal-head` / `.modal-x`（弹窗标题栏 + 关闭）样式；`.modal-card` 加 `max-height + overflow:auto`，长表单可滚。
2. `src/TZHJ.Gateway/Components/Shared/Modal.razor`（新增）：通用弹窗外壳。`Visible`/`VisibleChanged`（支持 `@bind-Visible`）、`Title`、`Width`、`CloseOnBackdrop`、`ChildContent`、`Footer`；点遮罩/✕ 关闭。与 [017](017-管理后台安全加固.md) 的确认弹窗 `ConfirmDialog` 并存（前者承载表单，后者承载确认）。
3. `src/TZHJ.Gateway/Components/_Imports.razor`：已含 `@using ...Components.Shared`（017 已加），`Modal` 直接可用。
4. `src/TZHJ.Gateway/Components/Pages/AdminUsers.razor`：去掉常驻"新建用户"卡片，页头加「+ 新建用户」；表单移入 `Modal`。`OpenCreate` 清空字段并开弹窗；`CreateUser` **成功才关弹窗 + 清空**，失败保留输入并在弹窗内顶部红条提示。
5. `src/TZHJ.Gateway/Components/Pages/AdminRoles.razor`：去掉常驻"新建/编辑角色"卡片，页头加「+ 新建角色」；表单（角色名/描述/数据范围/加范围）移入 `Modal`，标题随 `_editId` 在"新建/编辑"间切换。`OpenCreate`/`Edit` 均开弹窗；`CloseForm` 关弹窗并重置编辑态（✕/遮罩经 `OnFormVisibleChanged` 走同一路径）；`Save` **成功才关弹窗**，失败保留输入并在弹窗内提示。

不变：服务端 `AdminService` 及所有业务逻辑、权限模型、Cookie 鉴权、日志页、确认弹窗行为；无外部依赖。
