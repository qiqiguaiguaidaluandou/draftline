## 回传图纸核价中的价格到SRM中的接口

请求体：
```
{
    "interfaceCode": "AI_ITEM_PRICE_TO_SRM",
    "content": {
        "itemList": [
            {
                "itemCode": "110105067058",
                "price": "120"
            },
            {
                "itemCode": "110105131715",
                "price": "110"
            }
        ]
    }
}
```

interfaceCode是固定的字段
itemList是用于回传对应物料编码和价格的数组

响应：
```
{
  "responseDate": "2026-06-24T14:11:18.454",
  "msg": "操作成功",
  "code": "S",
  "status": "S",
  "data": [
    {
      "itemCode": "110105067058",
      "price": 120,
      "remark": "新增成功"
    },
    {
      "itemCode": "110105131715",
      "price": 110,
      "remark": "失败：物料编码在物料主数据中不存在"
    }
  ],
  "count": null,
  "page": null
}
```

回传价格的时候会出现三种情况：
```
{
    "status": "SUCCESS",
    "code": "200",
    "message": "操作成功",
    "data": [
        {
            "itemCode": "151633456",
            "price": 120.00,
            "remark": "更新成功"
        },
        {
            "itemCode": "78965",
            "price": 110.00,
            "remark": "新增成功"
        },
        {
            "itemCode": "99999",
            "price": 150.00,
            "remark": "失败：物料编码在物料主数据中不存在"
        }
    ]
}
```

这个接口只有在点击回传到SRM的按钮的时候才会调用
